🔀 Prediction Market Proxy

Serverless CORS proxy for Polymarket, Kalshi, SX Bet, and Manifold.

Routes

MethodPathEffect
GET/api/healthHealth check & provider list
GET/api/polymarket/markets?limit=100Polymarket CLOB markets
GET/api/kalshi/markets?limit=100Kalshi markets
GET/api/sxbet/marketsSX Bet markets
GET/api/manifold/markets?limit=50Manifold markets
GET/api/proxy?target=polymarket&path=/marketsExplicit param mode
POST/api/polymarket/orderPOST forwarded verbatim

Configuration

Env varDefaultPurpose
PROXY_TOKEN(unset)Require x-proxy-token header on every request
PROXY_TIMEOUT_MS30000Upstream timeout in ms
ALLOW_ANY_TARGETfalseAllow ?url=https://... direct proxying
POLYMARKET_BASE_URLhttps://clob.polymarket.comOverride Polymarket base URL
KALSHI_BASE_URLhttps://trading-api.kalshi.com/trade-api/v2Override Kalshi base URL
SXBET_BASE_URLhttps://api.sx.betOverride SX Bet base URL
MANIFOLD_BASE_URLhttps://api.manifold.markets/v0Override Manifold base URL

Deploy

Netlify: cd apps/proxyapp && netlify login && netlify init && netlify deploy --prod --dir=public --functions=netlify/functions

Vercel: cd apps/proxyapp && vercel login && vercel --prod

Full instructions: see HELP.md in the repo.
