# Trading Command Center Design System

## Visual Identity
Dark mode only, professional trading interface, high contrast, data dense dashboard.

## Colors
- Background: #0a0a0a (gray-950)
- Surfaces: #171717 (gray-900)
- Borders: #262626 (gray-800)
- Primary Accent: #34d399 (emerald-400)
- Danger: #f87171 (red-400)
- Warning: #fbbf24 (amber-400)
- Text Primary: #ffffff
- Text Secondary: #a3a3a3

## Typography
- Font: Inter
- Monospace: JetBrains Mono for all numeric values, timestamps, codes
- Font weights: 400 body, 500 labels, 600 headers

## Components
- Card corners: 0.75rem rounded
- Shadows: subtle elevation only for active states
- Status badges: solid background with 10% opacity, border 30% opacity

## Layout
- 16px base padding
- 24px card gutters
- 8px element spacing
- 1200px max container width

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Palette: Emerald (#34d399), Neutral Dark
- Styles: Soft rounded corners, subtle elevation, glassmorphism on active elements
